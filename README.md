  <h2>Ce este gazduirea web (hosting)?</h2>
  <p>Gazduirea web reprezintă serviciul prin care conținutul unui site web este stocat pe un server și făcut accesibil publicului prin intermediul internetului. În esență, când închiriezi un spațiu de găzduire, obții dreptul de a folosi resursele unui server pentru a-ți stoca fișierele site-ului, astfel încât acestea să fie disponibile utilizatorilor din întreaga lume, 24/7.</p>
  <p>Un server este, de fapt, un computer performant, configurat special pentru a funcționa continuu și a răspunde solicitărilor venite de la utilizatorii care accesează site-ul tău. Acesta conține toate fișierele, bazele de date și aplicațiile necesare pentru ca site-ul să funcționeze corect.</p>
